## Pattern matching générique avec un switch 
````csharp
 static int GetLength<T>(object items)
        {
            switch (items)
            {
                case Array array:
                    return array.Length;
                case List<T> list:
                    return list.Count;
                case IEnumerable<T> enumerable:
                    return enumerable.Count();
                default:
                    return 0;
            }
        }

void Sample()
{
    int[] i = new int[] { 1, 2, 3 };
    Console.WriteLine("Taille du tableau:" + GetLength<int>(i));
}
````
